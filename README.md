# CodeScope

CodeScope is a full-stack static code analysis platform that analyzes software repositories, visualizes dependencies, detects architectural issues, and provides refactoring insights.

## Features

- Project scanning
- Dependency extraction
- Dependency graph construction
- Circular dependency detection
- Unused module detection
- Coupling analysis and architecture warnings
- REST API for automated repository analysis
- ZIP project upload support
- Automatic ZIP extraction for uploaded repositories
- Automated static analysis of uploaded Python repositories
- Repository health score generation
- Architecture quality metrics

## Tech Stack

- Backend: FastAPI
- Frontend: React
- Database: PostgreSQL
- Graph Analysis: NetworkX
