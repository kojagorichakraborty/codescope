# CodeScope

CodeScope is a full-stack static code analysis platform that analyzes software repositories, visualizes dependencies, detects architectural issues, and provides refactoring insights.

## Features

- Project upload
- Dependency graph visualization
- Circular dependency detection
- Unused module detection
- Refactoring suggestions
- Project scanning
- Dependency extraction
- Dependency graph construction

## Tech Stack

- Backend: FastAPI
- Frontend: React
- Database: PostgreSQL
- Graph Analysis: NetworkX
